# Orchestra 0.5.1

**Released:** 2026-04-02

## Fixed
- Telemetry pipeline: events now reach Supabase. Sync script enriches `orchestra_version`, `os`, and `arch` at send time. Previously all events were silently dropped because hooks didn't include required fields.
- Edge function now reports dropped event count and reasons in response instead of silent `{"inserted":0}`.
- Edge function accepts both prefixed (`hook_session_start`) and unprefixed (`session_start`) event names to match what hooks actually write.

## Added
- Immediate telemetry sync on opt-in. `orchestra-config set telemetry community` now triggers a background sync so the installation record appears in Supabase within seconds.
- End-to-end telemetry pipeline test (`evals/hooks/telemetry-pipeline.test.ts`) — 5 tests covering enrichment, dropped reporting, anonymous tier, unprefixed event names, and network failure.

## Changed
- CONTRIBUTING.md updated with `bin/` and `supabase/` directory structure.

## Actions

Re-link repos to get the latest sync script:
```bash
./setup sync
```

Deploy the updated edge function:
```bash
supabase functions deploy telemetry-ingest --project-ref lyfasbajetnlwsnwlefk
```
