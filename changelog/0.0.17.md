# Orchestra 0.0.17

**Released:** 2026-03-23

## What's new

### Heartbeat context window fix (round 2)

The heartbeat had a compaction death spiral caused by multiple compounding bugs:

**Root causes identified:**
1. **Hooks triggered heartbeat → compaction loop.** Session-start hook said "Run `/o heartbeat` NOW." Post-compact hook dumped 500+ lines of context (full progress.yaml, full MEMORY.md, full daily logs). Combined, these pushed context right back to compaction threshold. After compaction, session-start's system-reminder persisted and re-triggered heartbeat. Infinite loop.

2. **Subagent used wrong cron prompt.** When cron scheduling was delegated to a subagent, the subagent simplified the inline prompt to `/o heartbeat` — creating a recursive scheduling bomb.

3. **Cron job leaks.** Cron jobs survive compaction but the agent's memory of them doesn't. Every compaction created duplicate cron jobs.

**Fixes:**
- **Hooks no longer trigger heartbeat.** Session-start and post-compact hooks inject context only — no "run `/o heartbeat`" instructions. Heartbeat is set up on the first `/o` dashboard invocation each session.
- **Post-compact hook is now lightweight.** Shows one-line summaries + file paths instead of dumping full file contents. ~15 lines of output instead of 500+.
- **Cron scheduling stays in main agent.** 3 deterministic tool calls (CronList → CronDelete all → CronCreate). Never delegated to subagents, which can't reliably construct the exact inline prompt.
- **Cron prompt is a minimal inline instruction**, never `/o heartbeat`. Prevents recursive skill invocation.
- **State audit file writes delegated to background subagents.** Main context only sees a one-line summary.

### Subagent delegation for heavy operations

`/o checkpoint`, `/o docs`, `/o heartbeat` state audit, and post-work audits now delegate file I/O to background subagents. The main context window is protected — it only sees one-line summaries. The `.orchestra/` files are the persistent log.

## Actions

Re-link repos to pick up the updated hooks:

```
action: relink
prompt: "Repos need re-linking to get the heartbeat fixes. Run setup sync now?"
command: __ORCHESTRA_DIR__/setup sync "$_ORCH_ROOT"
```
